# College Selection Report

React + TypeScript + Vite + Tailwind CSS project

## Setup Instructions

1. Install dependencies:

```bash
npm install
```

2. Run the development server:

```bash
npm run dev
```

The app will open at http://localhost:5173

## Build

```bash
npm run build
```

## Technologies

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Lucide Icons
